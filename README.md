# Movie Catalog Manager (Python)

This project is a command-line interface (CLI) application written in Python, designed to manage and search for information within a movie catalog. The data is dynamically loaded from a local text file. This is my first-ever project in Python, from my first year of studies.

## Features
* Structured reading and parsing from a text file (`Film.txt`).
* Multi-level interactive menu.
* Advanced search by: Title, Year (including year ranges), Actor, Genre, Country, Director, and Duration.
* "Random Movie" selection feature.

## Skills and Technologies implemented
In this project, I have implemented several key Python concepts:
* **Object-Oriented Programming (OOP):** Use of Classes, Objects, and special methods (Dunder methods like `__repr__`).
* **Data Structures:** Combined use of Lists, Dictionaries (used as hash maps to optimize search performance), and Sets (to handle unique elements).
* **File I/O:** Reading and processing external text data.
* **Control Flow & Algorithms:** Complex loops, string parsing and slicing, custom sorting logic, and recursive functions for menu navigation.
